JS判断当前浏览器是否启用falsh，并强制调用flash方法

JS判断当前浏览器是否启用falsh，并强制调用flash方法

前言

众所周知，falsh已经正式被Adobe公司所放弃，还记得乔布斯说过：“未来，将是html的天下，falsh终将被淘汰”；

谷歌浏览器一马当先，率先在谷歌浏览器当中默认禁止flash控件了，各大视频网站也纷纷抛弃flash，转向html5播放器，国内视频弹幕网站哔哩哔哩识得先机，早早的把自家的播放器升级成为了html5，最早尝到第一口科技带来的“蛋糕”，把国内许多知名大厂甩在身后。

诚然，flash被淘汰是迟早的事情，但是，不得不提国内还有许许多多的用户，或者是老的企业所用到的技术，还在停留在IE、flash，tab布局的上古时代，老板一句：“我要兼容IE6”，也不是没有可能发生的事情。

很不幸的事情就是，现就职的这家公司，就还在采用falsh技术作开发，利用falsh来作文件上传，本来也是老旧的东西，只要能用就行，这也无可厚非。可是昨天发生了一个bug就是让我感觉，这玩意还是赶紧淘汰吧。重构都来得好一些。。。2333333

问题产生

项目当中有一个功能就是网盘功能，大家都知道，它是用来存储数据的一个功能，必然会有一个上传的东西，我们是使用了flash作为上传控件，用的还是upload.js的一个控件，可以说是非常的老旧了，也就是这么个老古董，也有bug。

我们发现，当点击这个按钮的时候，根本没有执行选取文件这一步，也就说，点击上传选文件的时候，没有反应，用了360浏览器，反而可以（之前360被我狠狠的突出了一番）今天表现居然这么良好？

查找原因

后来查看代码看到，原来是浏览器的falsh插件并没有启动，导致这个控件根本没有执行。
可用下列代码获取当前浏览器的flash状态

    //判断是否开启了flash
flashChecker: function () {
    var hasFlash = 0; //是否安装了flash
    var flashVersion = 0; //flash版本
    var isIE = /*@cc_on!@*/ 0; //是否IE浏览器
    if (isIE) {
        var swf = new ActiveXObject('ShockwaveFlash.ShockwaveFlash');
        if (swf) {
            hasFlash = 1;
            VSwf = swf.GetVariable("$version");
            flashVersion = parseInt(VSwf.split(" ")[1].split(",")[0]);
        }
    } else {
        if (navigator.plugins && navigator.plugins.length > 0) {
            var swf = navigator.plugins["Shockwave Flash"];
            if (swf) {
                hasFlash = 1;
                var words = swf.description.split(" ");
                for (var i = 0; i < words.length; ++i) {
                    if (isNaN(parseInt(words[i]))) continue;
                    flashVersion = parseInt(words[i]);
                }
            }
        }
    }
    return {
        f: hasFlash,
        v: flashVersion
    };
},

知道原因就好办了，下面就让我们来解决这个bug

解决办法

现在，我们已经知道，产生这个bug的根本原因是因为当前浏览器没有启用flash，那我们利用JS代码来启用它不就行了吗？

首先，创建一个 a 标签

var a = document.createElement('a');

然后设置一堆属性

a.style.display = 'none';
a.href = "http://www.adobe.com/go/getflashplayer";
a.rel = "nofollow";
a.target = "_blank";

然后，添加到body里面去

document.body.appendChild(a);

最后，自动执行这个

a.click();

好了。最终效果就是这个样子。

var a = document.createElement('a');
a.style.display = 'none';
a.href = "http://www.adobe.com/go/getflashplayer";
a.rel = "nofollow";
a.target = "_blank";
document.body.appendChild(a);
a.click();

思路整理

首先定义一个函数，来判断是否开启了flash

//判断是否开启了flash
flashChecker: function () {
    var hasFlash = 0; //是否安装了flash
    var flashVersion = 0; //flash版本
    var isIE = /*@cc_on!@*/ 0; //是否IE浏览器
    if (isIE) {
        var swf = new ActiveXObject('ShockwaveFlash.ShockwaveFlash');
        if (swf) {
            hasFlash = 1;
            VSwf = swf.GetVariable("$version");
            flashVersion = parseInt(VSwf.split(" ")[1].split(",")[0]);
        }
    } else {
        if (navigator.plugins && navigator.plugins.length > 0) {
            var swf = navigator.plugins["Shockwave Flash"];
            if (swf) {
                hasFlash = 1;
                var words = swf.description.split(" ");
                for (var i = 0; i < words.length; ++i) {
                    if (isNaN(parseInt(words[i]))) continue;
                    flashVersion = parseInt(words[i]);
                }
            }
        }
    }
    return {
        f: hasFlash,
        v: flashVersion
    };
},

然后就是在点击的时候，首先执行这个函数，来获取当前的状态来判定，是否开启了flash，如果没有，则执行调用函数，否则，执行上传功能弹窗。

openAddFileDialog: function () {
var fls = netdisk.flashChecker();
if (fls.f) {
    $.dialog({
        id: 'nd01',
        title: '添加文件',
        content: $('#netdisk_uploadfile').html(),
        padding: '5px 25px',
        button: [{
            name: "上传",
            callback: function () {
                $('#uploadFile').uploadify('upload');
                return false;
            },
            focus: true
        }, {
            name: "取消上传",
            callback: function () {
                $('#uploadFile').uploadify('cancel');
                this.close();
                return false;
            },
        }],
        lock: true
    });
    netdisk.initUploadify();
} else {
        var a = document.createElement('a');
        a.style.display = 'none';
        a.href = "http://www.adobe.com/go/getflashplayer";
        a.rel = "nofollow";
        a.target = "_blank";
        document.body.appendChild(a);
        a.click();
    }
}

此代码是能够针对所有浏览器来判断是否开启，并且来调用flash，亲测chrome、Firefox、360、EDG、IE有效。
